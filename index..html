// 02 — Selected Work

Projects

PROJECT 01
2D Physics Shooter
Unity C# 2D Physics Particle Effects

A 2D Unity game built from scratch featuring a player with physics-based movement, a mouse-aimed turret, projectile shooting, spring pads, and firework particle effects on bullet impact.

  • Physics-based movement using Rigidbody2D with ground tangent detection for slope traversal
  • Mouse-aimed turret calculating rotation toward cursor using world-space vectors
  • Bullet physics with area-of-effect force and firework particle system on collision
  • Spring pad mechanic that launches the player on contact
  • Modular script design — each behaviour is a separate, reusable component
PROJECT 02
Top-Down Racing Game
Unity C# OOP Data-Driven Design

A top-down endless racer with player-controlled car movement, enemy vehicles, a fuel system, turbo boost, and bomb pickups. Designed with loose coupling and modular architecture.

  • WASD controls with Left Shift turbo boost affecting speed and road animation
  • Fuel system depleting over time; pickups restore fuel and animation
  • Enemy cars with randomised velocity and directional collision detection
  • Loose coupling via UnityEvents — no hard dependencies between components
  • All gameplay values exposed in the inspector for rapid iteration
▸ Play on itch.io
PROJECT 03
AI Police & Robbers
C++ Pathfinding State Machines BFS / DFS Maze Generation

A custom C++ game AI simulation built without a commercial engine. Cops and thieves navigate a procedurally generated maze — cops chase via state-machine AI, thieves steal diamonds and escape.

  • Procedurally generated maze using a seeded algorithm — unique layout every run
  • Cop AI with Wander, Chase, and Attack states transitioning on thief proximity
  • Thief AI pathfinding to diamonds while avoiding cops, tracking individual loot
  • BFS and DFS pathfinding implemented from scratch in C++
  • Fog-of-war exploration — agents only know cells they have visited
PROJECT 04
SG Tower Defence
Unity C# 2D Strategy Group Project

A Singapore-themed tower defence game built in Unity as a group project. Players defend against waves of enemies using strategically placed towers — with a local Singaporean twist on the classic genre. Playable in browser.

  • Singapore-themed setting, art, and design direction
  • Classic tower defence wave mechanics built in Unity
  • Developed collaboratively as a team project
  • Deployed as a browser-playable HTML5 game
